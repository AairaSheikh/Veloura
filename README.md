# Veloura Pilates Studio

> Mindful movement for a modern life

## Overview

Veloura is a modern, responsive website for a boutique Pilates studio located in Islamabad, DHA. The studio is dedicated to providing mindful, restorative, and empowering movement experiences for clients of all levels.

## About the Studio

Founded on the belief that movement should be **mindful, restorative, and empowering**, Veloura Pilates Studio creates a sanctuary where clients can pause, breathe, and reconnect with their bodies. The studio moves away from the "no pain, no gain" mentality and embraces sustainable, functional movement.

Whether you're recovering from an injury, looking to improve posture, or seeking a challenging athletic flow, Veloura's boutique space is designed for you.

## Features

### Studio Offerings

- **Reformer Flow** - A dynamic full-body workout using the Reformer to lengthen, strengthen, and tone. Focus on control, precision, and fluid transitions. (50 min, All Levels)

- **Mat Sculpt** - A high-energy mat class using props like balls, bands, and circles. Target deep core muscles and glutes. (45 min, Intermediate)

- **Private Sessions** - The ultimate personalized experience. Perfect for rehabilitation, pre/post-natal care, or mastering advanced techniques. (55 min, 1-on-1)

### Key Pages

- **Home** - Landing page with hero section and quick navigation
- **About** - Studio story and instructor profiles (including founder Sophie)
- **Classes** - Detailed class descriptions with filtering options
- **Schedule** - Class timetable and booking system
- **Pricing** - Membership and class packages
- **Contact** - Get in touch form and location information
